# Portfolio

This project is a personal portfolio page built with HTML5, CSS3, and responsive design.

## Overview

- Semantic HTML structure with `header`, `nav`, `main`, `section`, and `footer`
- Responsive layout using Flexbox and CSS Grid
- Mobile-first styles with breakpoints at `768px` and `1024px`
- CSS variables for easy theme control

## Files

- `index.html` — portfolio page markup
- `styles.css` — styling with responsive layout
